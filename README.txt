README Movie Website

To run this application open the entertainment_center.py file and create Movie instances.  Already included are Toy Story, Avatar, Happy Gilmore, School of Rock, Ratatouille, and Midnight in Paris. 

The paramters taken are:
1) Title
2) Starring Actor
3) Tag line
4) Poster image
5) Youtube URL for trailer (must use youtube)
6) Release Date
7) Duration
